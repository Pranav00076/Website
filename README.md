# DerronDie Open Source Community Website

## 🎯 Project Overview
A sleek, modern, single‑page website showcasing the **DerronDie** open‑source community. Built with **Tailwind CSS**, **HTML**, and a touch of JavaScript to provide a smooth scrolling experience, active navigation highlighting, and an engaging UI that follows the latest design trends (glassmorphism, dark mode, vibrant gradients).

---

## ✨ Key Features
- **Responsive, mobile‑first layout** with fluid grids and container queries.
- **Dynamic navigation bar**:
  - Active link highlighting on click and scroll.
  - Fixed header with smooth scroll‑margin to prevent content overlap.
- **Contributor showcase** with live avatars pulled from GitHub.
- **Community feed placeholder** ready for future integration.
- **Tailwind CSS + custom design tokens** for a premium aesthetic (dark mode, glowing hover effects, pulse animations).
- **Zero‑JS dependencies** – only vanilla JavaScript for interactivity.

---

## 🛠️ Installation & Development
```bash
# Clone the repo
git clone https://github.com/Demon-Die/Website.git
cd Website

# No build step – open locally
open index.html   # macOS
# or serve with any static server
npx -y serve .   # quick local dev server
```
The site relies on the Tailwind CDN, so no npm install is required.

---

## ▶️ Usage
- Open `index.html` in a browser.
- The navigation links (`Home`, `Projects`, `Contributors`, `Community`, `About`) scroll smoothly to their sections.
- Click the **“Join the contributors”** card to open the GitHub organization members page.
